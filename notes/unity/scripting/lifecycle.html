MonoBehaviour生命周期

返回目录
脚本生命周期方法在特定时间由Unity自动调用，理解这些方法的执行顺序至关重要 
using UnityEngine;

public class ExampleScript : MonoBehaviour
{
    // 初始化(脚本启用时调用)
    void Awake() { /* ... */ }
    
    // 开始(第一帧更新前调用)
    void Start() { /* ... */ }
    
    // 每帧更新
    void Update() { /* ... */ }
    
    // 固定时间步长更新(用于物理计算)
    void FixedUpdate() { /* ... */ }
}
方法 调用时机
Awake 脚本实例被创建时调用，无论脚本是否启用
OnEnable 脚本启用时调用
Start 在Update第一次调用前执行
Update 每帧调用一次
LateUpdate 在所有Update调用后执行
OnDisable 脚本禁用时调用
OnDestroy 对象销毁时调用
